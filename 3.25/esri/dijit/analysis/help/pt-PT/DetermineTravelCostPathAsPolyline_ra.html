Determinar Percurso de Custo de Viagem Como Polilinha

This tool calculates the least cost polyline path between sources and known destinations.

Como exemplos de aplicações, incluem-se as seguintes:

  • Encontre o melhor percurso para uma nova estrada em termos de custos de construção.
  • Identifique o percurso a tomar a partir de vários locais suburbanos (origens) até ao centro comercial mais próximo (destino).

Se a opção Utilizar a extensão atual do mapa se encontrar selecionada, apenas os locais nas camadas de entrada que são visíveis na extensão atual do mapa serão analisados. Se não selecionar a opção, serão analisados todos os locais em ambas as camadas de entrada, mesmo que se encontrem fora da extensão atual do mapa.

Selecionar camada raster ou de camada de origem a partir da qual determinar percursos

Um serviço de imagens ou serviço de elementos que identifica as células a partir das quais o percurso de menor custo é determinado para o destino de menor custo.

Caso a entrada seja um serviço de imagem, a entrada consiste de células que têm valores válidos (zero é um valor válido), e às células restantes tem de ser atribuído o valor NoData.

Selecionar a camada raster de custo de viagem

A raster defining the impedance or cost to move planimetrically through each cell.

The value at each cell location represents the cost-per-unit distance for moving through the cell. Each cell location value is multiplied by the cell resolution while also compensating for diagonal movement to obtain the total cost of passing through the cell.

The values of the cost raster can be integer or floating point, but they cannot be negative or zero (you cannot have a negative or zero cost).

Raster ou Elementos de Destino de Entrada

Uma entrada que define os destinos utilizados para calcular a distância. A camada pode ser de dados ou de elementos.

Caso a entrada seja um serviço de imagem, a entrada consiste de células que têm valores válidos (zero é um valor válido), e às células restantes tem de ser atribuído o valor NoData.

Nome da camada de polilinhas resultante

The name of the layer that will be created in My Content and added to the map. The default name is based on the tool name and the input layer name. If the layer already exists, you will be asked to provide another name.

Using the Save result in drop-down box, you can specify the name of a folder in My Content where the result will be saved.

Selecionar tipo de percursos a determinar

Define o modo como os valores e zonas no destino de entrada serão interpretados nos cálculos de custo de percurso

  • Melhor Único—Para todas as células ou locais nos dados de destino de entrada, o percurso de menor custo é derivado do local com o mínimo de percursos de menor custo até ao local de origem. Esta é a predefinição.
  • Para Cada Célula—Por cada célula ou local com valores válidos nos dados de destino de entrada, é determinado e guardado um percurso de menor custo no serviço de elementos de saída. Com esta opção, cada célula dos dados de destino de entrada é tratada separadamente, e é determinado um percurso de menor custo para cada célula.
  • Para Cada Zona—Por cada zona nos dados de destino de entrada, é determinado e guardado um percurso de menor custo no serviço de elementos de saída. Com esta opção, o caminho de menor custo para cada zona começa na célula ou local com a menor ponderação de custo e distância na zona.
